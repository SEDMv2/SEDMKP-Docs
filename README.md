# SEDMKP-Docs
SEDM v2 (Kitt Peak) Documentation

Requirements:
1. sphinx 4.1.2
2. Alabaster 0.7.12

Procedure:
- edit files in source directory
- update version date in conf.py
- cd ..
- make html
- cd build/html
- open index.html
  - verify changes
- update repository
  - git add <changes>
  - git commit -m "<notes>"
  - git push 
- Notify deployment via email
